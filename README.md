# Team Admin Portal - Workspace Africa

Next.js frontend for Team Admin portal of Workspace Africa.

## Features

- **Dashboard**: Team overview with statistics
- **Member Management**: Invite, view, and remove team members
- **Billing & Subscription**: Manage team plans and payments
- **Analytics**: Team usage insights and reports
- **Settings**: Team configuration and preferences

## Tech Stack

- **Framework**: Next.js with Pages Router
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **HTTP Client**: Axios
- **Icons**: Lucide React

## Getting Started

1. Install dependencies: `npm install`
2. Set environment variables in `.env.local`:

## Getting Started

1. Install dependencies: `npm install`
2. Set environment variables in `.env.local`:NEXT_PUBLIC_API_URL=https://workspace-africa-backend.onrender.com
3. Run development server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Project Structure

- `/pages` - Next.js pages (dashboard, members, billing, analytics, settings)
- `/components` - Reusable React components
- `/components/ui` - shadcn/ui components
- `/lib` - Utility functions and API client
- `/styles` - Global styles and Tailwind config

## Deployment

Deployed on [Vercel](https://vercel.com) or similar platform.
